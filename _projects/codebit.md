[**CodeBit**](https://codebit.xyz) is an online app to generate animated code videos. The kind of videos you can post on Twitter or LinkedIn to show a quick refactoring, how to use a library, or how to fix a bug.

I created this project with [Maxime](https://maxday.dev). We loved these videos but never found the right tool to create them without spending long hours tweaking transitions or syntax highlighting.
